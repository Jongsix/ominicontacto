===================================
Omnileads (OML) 1.5.3 release notes
===================================

*April 3, 2020*


This is a "bugfix" release improving the stability and performance of the codebase.


Fixes and improvements
=========================
- Fixed installation error on database creation on cloud environments
- Fixed error on release update status information
- Fixed inconsistency on log events generated when agent refresh the page
- Fixed error on inbound calls supervision report when there are only transfer calls in the campaign in the current day
- Fixed error on logout sincronization between Django/Asterisk
