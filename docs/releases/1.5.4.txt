===================================
Omnileads (OML) 1.5.4 release notes
===================================

*April 8, 2020*


This is a "bugfix" release improving the stability and performance of the codebase.


Fixes and improvements
=========================
- Fixed installation error when upgrading to release-1.5.3
- Fixed error on engaged call disposition updating process that generates a campaigns dispositions report error
- Fixed error on exporting to csv of inbound campaigns disposition report that showed more than one record for a single call disposition
