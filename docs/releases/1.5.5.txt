===================================
Omnileads (OML) 1.5.5 release notes
===================================

*April 14, 2020*


This is a "bugfix" release improving the stability and performance of the codebase.


Fixes and improvements
=========================
- Added improvement to the supervision that only shows information about active agents asigned to the logged supervisor
- Added improvement that includes call duration in contacts called report when exporting to .csv file
- Added improvement that warns to supervisor users when they attempts to left the active agents page and are in a middle of Asterisk action, such as spy, whisper, etc
- Fixed error on file format when exporting to .csv and open exported file with Microsoft Excel, when user entered end line characters on disposition comments
